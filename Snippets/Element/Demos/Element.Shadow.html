Element Shadow

Introduction

let you easily create shadows for elements

Default Shadow

works also for divs... :)
it's as simple as $('myElement').shadow();
Notice: you need to set a background-color for divs!
Notice: you can't use padding on the shadow div (if you need padding you have to wrap it in another div)
