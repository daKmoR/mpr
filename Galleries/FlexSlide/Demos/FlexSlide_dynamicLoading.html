FlexSlide.Advanced dynamic Loading

Introduction

With FlexSlide.Advanced you can dynamically load images, urls, inline content

Example for dynamic Loading of some images from flickr

So now this is awesome. This gallery contains 30 images from flicker however they all will be loaded on demand. So if you open up this page not a single image from flickr is trasfered. All the gallery items are links which will be dynmically loaded. You can load stuff like images, request and inline content. Once the image is loaded once it will be cached and reused so you don't have to wait every time.

I'm just loving it :)
