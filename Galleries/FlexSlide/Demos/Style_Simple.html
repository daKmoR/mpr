DIV
This is a simple text div containing some inline style.
It demonstrates the autoItemSize feature for width and height. It won't be centered - yeah it has the full size so it won't work.
However it could be done by setting
autoItemSize: { x: true, y: false } (which is default) and
centerItemTags: ['img', 'a', 'div'].
Using this the div would have the full width and would be centered vertically.
